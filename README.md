# AdoptionCenterDA 🐾

## Project Description

**AdoptionCenterDA** is a .NET Blazor web application designed to help users discover pets available for adoption and connect with adoption services.

The goal of this project is to create a user-friendly platform where people can view available pets, learn more about them, and submit adoption requests. The application will focus on usability, accessibility, and a clean user experience while following modern software development practices.

This project is developed as part of the .NET Development course using the Blazor framework and the .NET technology ecosystem.
